word2vec
========

Python interface to Google word2vec

Installation
------------

`pip install word2vec`

Use [virtualenvs](http://www.virtualenv.org/en/latest/)!

Usage
-----

The default functionality from google is available:
* word2vec
* word2phrase
* w2v-distance
* w2v-word-analogy
* w2v-compute-accuracy
